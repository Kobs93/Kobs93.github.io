L'APCIS (Association Accueils, Préventions, Cultures : Intercommunautaires et Solidaire) est une association qui a été crée en 1990 dans la ville de Stains dont le but est de répondre aux besoins des habitants de la ville.

C'est une association subventionnée par l'Etat et elle est composée de 10 employés qui sont chargée d'aider les habitants de la ville. D'autres antennes ont été ouvertes à Bagnolet et Epinay-Sur-Seine.


L'association :

  • Forme juridique : Association
  • Effectif : 10
  • Location : 10 Rue Guillaume apolinaire 93240 Stains.li>

Missions:

  • Création d'un site vitrine
  • Participer aux réunions pour proposer des sujets d'articles pour le site.

Définition des ressources matérielles :

  • Windows 11 pro
  • Figma
  • Wordpress

Cahier des charges:

  • Site vitrine pour que les habitants de la ville puissent s'informer sur les services proposé par l'association.
  • Création d'une page d'actualité pour informer les étudiants des activités de l'association
  • Page regroupant les dispositifs de l'association
  • Page de présentation de l'équipe
  • liens vers les différents antennes

RGPD :

L'association ne possède pas de RGPD car le site vitrine ne récolte pas les données à caractère personnelle des utilisateurs.

OrangeExpress est une entreprise crée en 2003 par Monsieur Fabrice Meimoun, elle consiste dans la relation inter-entreprise en vendant des machines d'alimentations pour les entreprises comme des presse agrumes, frigos, machine à café, etc...

L'entreprise :

  • Dénomination : OrangeExptrss
  • Forme juridique : SAS (Société par action simplifié)
  • Effectifs : 1 à 2 employés
  • Taille : TPE (Très Petite Entreprise)
  • Secteur d'activité : Commerce de gros d'équipement pour le commerce, Commerce interentreprises
  • Capital social : 9 000 euros
  • Location : 59 Rue Des petits champs 75001 Paris

Mission :

Iln nous a été confié la réalisation d'un audit de code pour le site shop.fastconcepts.fr dans lequel nous étions chargé d'analyser le code du site web afin de fournir un rapport détaillé sur les différents problèmes de sécurité, de vérifier la qualité du code et d'autre problèmes pour mettre le site en conformité avec la loi.


Définitions des ressources matérielles et logicielles :

  • Ordinateur portable
  • Symfony
  • Docker
  • Git Hub
  • Notion
  • Canva

Réalisation de l'audit :

Site Web : Fastconcept

  • Le projet était sur un Git Hub personnel et nom propre à l'organisation
  • Il manquait l'historique du projet
  • Les commits ont été effectué le même jour

Sécurité :

  • Impossibilité de changer ses informations personnelles
  • Les mails arrivaient directement dans les spams
  • Pas de bot recaptcha dans le formulaire de contact
  • Accès aux devis des clients grâce aux URL

RGPD

  • Pas de page de politique de confidentialité
  • Absence à la norme du commerce éléctronique
